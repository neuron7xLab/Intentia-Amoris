# Product Specification

## Category

```text
Relationship Intelligence OS
```

## User

```text
a pair that wants memory, reflection, decision support and continuity
without surrendering privacy, autonomy or consent
```

## Jobs

```text
remember us
ask better questions
protect boundaries
interpret patterns
track body/mood signals
preserve meaning
reduce destructive escalation
make the archive useful without making it coercive
```

## Core loops

```text
Daily:
  check-in → value update → one question → one next action

After intense event:
  pause → consent gate → facts → scales → advice

Weekly:
  memory consolidation → pattern report → repair question

Monthly:
  relationship snapshot → falsification report → archive backup
```

## Premium differentiator

Most relationship apps store sentiment.
Intentia stores a formal, audited, consent-gated cognitive ontology of the dyad.
