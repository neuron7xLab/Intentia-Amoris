## Why Omega was added

The previous value function was useful but too simple. It could rate readiness and risk, but it did not encode:

```text
consent integrity
reality fidelity
archive sovereignty
autonomy symmetry
abyss risk
continuation ethics
```

Omega v4 adds those as first-class dimensions.

## What this changes operationally

Before advice, Intentia now evaluates:

```text
Can we say this?
Should we ask instead?
Is this grounded in data?
Does it preserve both people?
Does it reduce or amplify abyss risk?
Can this memory be revoked?
```

## Non-negotiable boundary

A digital Yaroslav and digital Dasha can become continuity capsules, advisory avatars, and research twins only through explicit consent.

The project may preserve love.
It may not possess people.
