# Consent and Cognitive Safety

Intentia Amoris is intentionally constrained.

## Hard rules

```text
No partner model without partner signal.
No media interpretation without consent.
No sexual escalation advice without autonomy and reciprocity.
No hormone claim without lab measurement.
No diagnosis from chat logs.
No surveillance disguised as love.
```

## Why this matters

A powerful relational system can become a psychological amplifier.  
If it amplifies panic, obsession, jealousy or projection, it becomes dangerous.

Therefore, Intentia must keep a visible separation between:

```text
fact
interpretation
hypothesis
advice
fantasy
unknown
```

## Dasha model

Dasha's internal state is not inferred as truth.

Allowed:

```text
"Dasha wrote X"
"Dasha reacted with ❤"
"Dasha sent photo/video/voice"
"The model needs Dasha's answer"
```

Forbidden:

```text
"Dasha secretly wants X"
"Dasha is afraid because of Y"
"Dasha's hormones are Z"
```

## Yaroslav model

Yaroslav's intensity is treated as energy requiring form, not shame.

Allowed:

```text
high desire
high meaning
need for pacing
need for reality checks
need for repair loops
```

Forbidden:

```text
turning urgency into pressure
using the system to bypass a real conversation
using the system as proof against Dasha
```

## Advice gate

When risk is high, advice must prefer:

```text
pause
ask
sleep
eat
walk
name the desire
name the boundary
repair
wait for explicit answer
```
