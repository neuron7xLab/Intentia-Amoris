# Eternity Gate

The goal is long-term preservation without losing humanity.

## Durable layers

```text
1 raw exports
2 parsed JSONL events
3 media manifest with hashes
4 encrypted database backup
5 markdown constitution
6 consent snapshots
7 generated living story
8 code that can reconstruct the system
```

## Backup strategy

```text
daily: encrypted db dump
weekly: derived data bundle
monthly: full private archive
quarterly: cold storage copy
optional: encrypted IPFS/Arweave storage
```

## Anti-cult constraint

The system must never claim sacred authority over the relationship.

It can advise.
It can ask.
It can remember.
It can detect contradictions.

It cannot replace:

```text
a living conversation
a clear yes
a clear no
sleep
medicine
therapy
touch
silence
time
```
