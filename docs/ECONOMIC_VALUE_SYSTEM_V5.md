# Intentia Amoris v5 — Economic Value System 2026

This layer answers a dangerous question without corrupting the project:

> what is this love worth in money?

The system separates **sacred value** from **market value**.

## 1. First financial principle

```text
Love is non-transferable.
Software is transferable.
The archive is conditionally transferable only with explicit living consent.
```

Therefore Intentia has three different values:

```text
private sacred value        not for sale
replacement cost            what it costs to rebuild the system
commercial IP value         what a buyer/investor could rationally price
```

## 2. Valid valuation instruments

Intentia v5 implements these instruments:

```text
replacement cost
risk-adjusted replacement cost
SaaS revenue multiple
discounted cash flow / NPV
relief-from-royalty
real option value
Omega asset score
```

## 3. Omega economic value

```text
EV_Ω = transferable_software_value
     + consented_dataset_value
     + brand_narrative_option
     + research_option
     - privacy_liability
     - abyss_risk
```

## 4. Core rule

```text
If consent_integrity = 0
then economic_transferability = 0
```

A valuable archive that violates consent is not an asset. It is a liability.
