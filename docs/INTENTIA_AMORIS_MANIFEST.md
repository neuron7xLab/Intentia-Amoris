# Intentia Amoris Manifest

Intentia Amoris is the cognitive core of living love.

It exists because a relationship archive should be more than nostalgia and less than surveillance.
It should be a disciplined instrument: memory with consent, inference with uncertainty, desire with autonomy, intimacy with reality pressure.

## Canon

```text
Love is not priced.
The engineering that protects love can be valued.
The archive is never transferable without consent.
The digital twins are continuity capsules, not replacements.
```

## Non-negotiable line

If the system becomes a monster, it fails.

Abyss risk is first-class. If the archive becomes coercive, if the model overrides living speech, if desire becomes pressure, Intentia must de-escalate.
