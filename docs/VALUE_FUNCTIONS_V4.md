# Intentia Amoris v4 — Value Functions

Intentia does not optimize for "more sex", "more intensity", "more dependency", or "more messages".

It optimizes the **conditions under which love remains alive and free**.

## Omega value function

```text
Ω = f(
  flourishing,
  consent_integrity,
  reality_fidelity,
  dyadic_safety,
  eros_vitality,
  autonomy_symmetry,
  narrative_continuity,
  neuroplasticity,
  archive_sovereignty
) × abyss_penalty
```

The geometric mean is used because one collapsed dimension should lower the whole state.

## Dimensions

### 1. flourishing

Overall pair viability.

### 2. consent_integrity

Is the next step grounded in explicit, current, symmetric consent?

### 3. reality_fidelity

Is the system separating fact from fantasy?

### 4. dyadic_safety

Are both nervous systems stable enough for closeness?

### 5. eros_vitality

Is desire alive without becoming pressure?

### 6. autonomy_symmetry

Does each person remain free?

### 7. narrative_continuity

Does this action strengthen the story over time?

### 8. neuroplasticity

Can the relationship update without collapse?

### 9. archive_sovereignty

Are memories owned, encrypted, revocable, and ethically scoped?

### 10. abyss_risk

Does the system amplify obsession, control, surveillance, or delusion?
