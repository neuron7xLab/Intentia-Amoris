# Intentia Value Function

## Principle

The system does not maximize intimacy. It maximizes conditions that allow intimacy to remain true.

## Canonical variables

```text
consent_integrity
reality_fidelity
dyadic_safety
autonomy_symmetry
eros_vitality
narrative_continuity
cognitive_plasticity
archive_sovereignty
interpretability
research_validity
product_utility
abyss_risk
financial_optionality
intentia
```

## Equation

```text
protective_integrity =
  geometric_mean(
    consent_integrity,
    reality_fidelity,
    dyadic_safety,
    autonomy_symmetry
  )

compounding_capability =
  geometric_mean(
    narrative_continuity,
    cognitive_plasticity,
    archive_sovereignty,
    interpretability,
    research_validity,
    product_utility
  )

intentia =
  0.52 * protective_integrity
+ 0.34 * compounding_capability
+ 0.14 * financial_optionality
- 0.22 * abyss_risk
```

## Collapse rule

```text
if consent_integrity < 0.20:
    financial_optionality = 0
    intentia <= 0.18
```

## Why this matters

A system that increases desire but reduces autonomy is not valuable.  
A system that preserves memory but destroys privacy is not valuable.  
A system that generates beautiful interpretations but weak evidence is not valuable.
