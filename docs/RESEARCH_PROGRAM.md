# Research Program

Intentia Amoris can become a research artifact only if it is falsifiable.

## Hypotheses

```text
H1: Consent-first prompting reduces escalation under high urgency.
H2: Event-sourced memory improves relationship repair recall.
H3: Scenario extrapolation improves decision clarity versus direct advice.
H4: Multimodal archives increase narrative continuity when consent is explicit.
H5: Abyss-risk metrics reduce coercive use of private archives.
```

## Evaluation

```text
clarity_delta
repair_success_rate
consent_violation_count
advice_acceptance_quality
false_claim_rate
partner_correction_rate
privacy_incident_count
retention_of_meaning
```

## Baselines

```text
free chat conversation
static diary
ordinary RAG notebook
ordinary couple app
manual reflection journal
```

## Falsification examples

```text
if false_claim_rate rises after adding VLM captions → multimodal interpretation is too aggressive
if partner_correction_rate rises after advice → advice overreaches
if consent_violation_count > 0 → release is blocked
if abyss_risk rises with usage → product is psychologically unsafe
```
