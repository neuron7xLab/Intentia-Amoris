# Golden state: secure closeness

Безпечна близькість у цій системі означає:

- обидва можуть сказати так;
- обидва можуть сказати ні;
- бажання не зникає, але не тисне;
- дотик відчувається як запрошення, не захоплення;
- слова уточнюють тіло;
- тіло не мусить захищатися;
- майбутнє не використовується як аргумент для примусу.
